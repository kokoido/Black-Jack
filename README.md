# Blackjack Game

A simple, visually styled **Blackjack** game built with **HTML, CSS, and JavaScript**. Players can place bets, hit, stand, or double their cards while tracking their bank, total gains, and losses.  

#Screenshots

https://github.com/kokoido/Black-Jack/issues/1#issue-3870069108

## Features

- Dynamic **player and dealer scores**
- Player actions: `Hit`, `Stand`, `Double`
- **Betting system** with bank management
- Track **win rate, total gains, and losses**
- Simple and clean **user interface**
- Fully responsive for modern browsers

## Gameplay

1. Place a bet.
2. Receive two cards.
3. Decide whether to **Hit**, **Stand**, or **Double**.
4. Dealer plays automatically.
5. Results update **bank, total gains/losses, and win rate**.

### Controls

- **Hit:** Draw another card.
- **Stand:** Keep current hand and let the dealer play.
- **Double:** Double the bet, draw one final card, and stand.
